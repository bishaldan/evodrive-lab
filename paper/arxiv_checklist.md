# arXiv Submission Checklist

## Authorship and ownership

- Confirm all listed authors approved the manuscript.
- Confirm no AI system is listed as an author.
- Confirm all figures, tables, and text are owned or properly attributed.

## Submission readiness

- Confirm the manuscript matches the public repository state.
- Confirm the abstract, introduction, and methods are consistent with the implemented benchmark.
- Confirm the paper does not overclaim the strength of `numpy_ppo_lite`.

## arXiv policy checks

- Check whether endorsement is required for the submission category.
- Choose the submission license intentionally and record it.
- Check for text overlap against README text, prior posts, or reused manuscript fragments.
- Add a note about significant AI-assisted writing or editing if required by field norms.

## Technical checks

- Build the LaTeX sources cleanly.
- Ensure all included figures exist and compile correctly.
- Ensure all bibliography entries resolve.
- Verify PDF metadata includes the correct title and author names.

## Repository checks

- Tag the paper-supporting commit.
- Ensure experiment scripts regenerate the main tables and figures.
- Ensure generated results referenced in the manuscript are reproducible from code.
